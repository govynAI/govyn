**Create `src/cli.ts`:** This is the unified CLI entry point.
- Add `#!/usr/bin/env node` shebang as first line (for npx execution).
- Parse `process.argv` for subcommands:
  - `govyn init` -> call the init wizard from `./init-wizard.js`
  - `govyn` (no args) or `govyn start` -> load config and start server (reuse logic from `src/index.ts`)
  - `govyn --help` -> print usage: `Usage: govyn [command]\n\nCommands:\n  start   Start the proxy server (default)\n  init    Interactive setup wizard\n\nOptions:\n  --config <path>  Path to config file (default: govyn.config.yaml)\n  --help           Show this help message\n  --version        Show version number`
  - `govyn --version` -> read version from package.json and print it
- For the `start` command: extract `--config` flag from argv. Call `loadConfig(configPath)`. Create CostAggregator, BudgetEnforcer, LoopDetector, ActionLogger just like `src/index.ts` does. Call `startServer(...)`. This makes `src/cli.ts` the new canonical entry, while `src/index.ts` remains as the library/programmatic entry point.

**Create `src/init-wizard.ts`:** Interactive wizard using Node.js built-in `readline` module (zero new dependencies).
- Export an async function `runInitWizard()`.
- Create a readline interface with `createInterface({ input: process.stdin, output: process.stdout })`.
- Ask questions in sequence using a helper `ask(question: string): Promise<string>`:
  1. "Which LLM providers do you want to use? (comma-separated: openai, anthropic, custom)" -> parse into array
  2. For each selected provider:
     - If openai: "Enter your OpenAI API key (or press Enter to use OPENAI_API_KEY env var):" -> if provided, validate it starts with `sk-`; store the env var name
     - If anthropic: "Enter your Anthropic API key (or press Enter to use ANTHROPIC_API_KEY env var):" -> if provided, validate it starts with `sk-ant-`; store the env var name
     - If custom: "Enter the custom endpoint name:" and "Enter the base URL:" and "Enter the API key env var name (or press Enter for none):"
  3. "Do you want to set a daily budget limit? (Enter amount in USD, or press Enter to skip):" -> parse as float
  4. "Enter a name for your first agent (default: 'default-agent'):" -> default to 'default-agent'
- Build a YAML config string from the answers using the `yaml` library's `stringify()` function. Use the same schema as `govyn.config.yaml`.
- Write the config to `./govyn.config.yaml` in the current working directory.
- Print: "Config written to ./govyn.config.yaml"
- Print: "Start the proxy with: npx govyn"
- Print a curl example: `curl http://localhost:4000/health`
- Close the readline interface.
- Do NOT store API keys in the config file directly. Instead, store the env var name (e.g., `api_key_env: OPENAI_API_KEY`). If the user provided an actual key, print a reminder: "Set your API key as an environment variable: export OPENAI_API_KEY=sk-..."
